# AES-256 Encryption Tool

A simple file encryption/decryption tool using AES-256 with a GUI built in Python.

## Features
- Encrypt any file using AES-256
- Decrypt with matching key file
- Simple and user-friendly GUI

## Requirements
- Python 3.x
- pycryptodome
- tkinter
